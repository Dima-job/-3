# MyBlogApp — Backend на FastAPI

Приложение для блога с расширенным функционалом.

## Реализованные функции:

- Регистрация пользователей
- Создание, редактирование и удаление постов
- Система подписок на авторов
- Приватные посты "только по запросу"
- Комментарии и теги

## Как запустить:

1. Активировать venv: `.\venv\Scripts\activate`
2. Запустить сервер: `uvicorn main:app --reload`
3. Открыть документацию: `http://127.0.0.1:8000/docs`
